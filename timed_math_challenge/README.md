# Math Quiz Game

A simple and fun math quiz game to test your basic arithmetic skills! The game generates 10 random math problems involving addition, subtraction, and multiplication. You need to answer each question as fast and correctly as possible. The game tracks the number of correct and incorrect answers and measures your total time to complete all the problems.

---

## Features

- **Random Math Problems**: The game generates random problems with operands between 3 and 12.
- **Operators**: The problems involve the following operators: addition (+), subtraction (-), and multiplication (*).
- **Timer**: Your time to complete the quiz is tracked.
- **Score**: Tracks both the number of correct and incorrect answers.
- **Try Again**: If you answer incorrectly, the game prompts you to try again until you get the right answer.

---

## How to Play

1. Run the script in your terminal or IDE.
2. Press Enter to start the quiz.
3. The game will present you with 10 random math problems. Type your answer and press Enter.
4. If your answer is correct, the game moves on to the next problem. If incorrect, you will be prompted to try again.
5. Once all problems are solved, the game will show your total time and your score.

---

---

## Requirements

- Python 3.x (for running the script).

---

## How to Run

1. Save the script as `math_quiz.py`.
2. Open your terminal or IDE and navigate to the folder where the file is saved.
3. Run the script using:
   ```bash
   python math_quiz.py
   ```

---

## Notes

- You can modify the range of operands by adjusting `MIN_OPERAND` and `MAX_OPERAND` in the script.
- The total number of problems can be changed by modifying the `TOTAL_PROBLEMS` variable.
- The time it takes to complete the quiz is displayed in seconds.
